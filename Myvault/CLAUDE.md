# Newvault — Claude Code Context

## Project Overview
This is a personal Obsidian knowledge management vault (PKM) for organizing notes, projects, and research.

## Vault Structure
```
Newvault/
├── 00_Inbox/          # Quick capture, weekly review
├── 01_Daily/          # Daily notes (YYYY-MM-DD.md)
├── 02_Projects/       # Active projects (Web Dev, Automotive, Home & Garden)
├── 03_Work/           # Work research, meeting notes, resources
├── 04_Personal/       # Health, hobbies, travel, finance, home management
├── 05_Knowledge/      # MOC (Map of Content) reference notes
├── 06_Archive/        # Completed projects
├── 07_Templates/      # Note templates
├── 08_Attachments/    # Photos, PDFs, documents
├── _Meta/             # Vault index, dashboard, plugin list
└── Myvault/           # Claude AI files and configuration
```

## Naming Conventions
- Daily notes: `YYYY-MM-DD.md`
- Project notes: `PascalCase_With_Underscores.md`
- Templates: `*_Template.md`
- Archive entries: `Archive_YYYY-MM-DD_Name.md`

## Tagging System
- `#project` — active project
- `#work` — work-related
- `#personal` — personal
- `#urgent` — high priority
- `#idea` — brainstorm
- `#research` — research topic
- `#archived` — archived content

## Languages
Notes are written in Turkish and English. Turkish is preferred for personal/daily notes.

## Key Files
- `_Meta/Index.md` — vault entry point
- `_Meta/Daily_Dashboard.md` — daily overview
- `_Meta/PKM_System.md` — system documentation
- `07_Templates/` — reusable note templates

## When Editing This Vault
- Preserve existing folder structure
- Follow the naming conventions above
- Use Obsidian internal links: `[[Note_Name]]`
- Tags go in frontmatter or inline with `#tag`
