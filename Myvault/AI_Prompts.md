# AI Prompts & Claude Workflows

## Quick Note Generation

### Daily Note
```
Create a daily note for [DATE] following the vault template:
- Top priorities
- Notes/thoughts
- What I learned
- Reflection
```

### Project Note
```
Create a project note for "[PROJECT NAME]" with:
- Status, start date, deadline, tags
- Clear objective
- Milestone checklist
- Resources section
```

### Research Note
```
Summarize [TOPIC] into a structured Obsidian note with:
- Key concepts
- Sources
- Related vault notes (suggest [[links]])
- Action items
```

## Vault Maintenance

### Weekly Review Prompt
```
Review my 00_Inbox notes and suggest:
1. Which notes to move to which folders
2. Notes that need linking to existing content
3. Tasks to add to active projects
```

### MOC Update Prompt
```
Update MOC_[TOPIC].md with new notes added this week.
Suggest new sections if needed.
```

## Writing Assistance

### Note Refinement
```
Improve this Obsidian note: keep it atomic (one idea),
add relevant [[wikilinks]], suggest tags, improve structure.
```

### Zettelkasten-style Note
```
Convert this raw text into a permanent note:
- Atomic single idea
- In my own words
- Linked to related concepts
- Timestamped title
```
